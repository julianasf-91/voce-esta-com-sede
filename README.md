# 💧 Você está com sede? — Projeto JavaScript

Projeto simples desenvolvido para **praticar conceitos básicos de JavaScript**, manipulando elementos da página a partir de interações do usuário.

A aplicação permite alterar a imagem de um copo conforme o nível de sede selecionado pelo usuário.

---

## 🚀 Funcionalidades

- Alteração dinâmica da imagem do copo
- Interação com botões usando **eventos JavaScript**
- Efeitos visuais ao passar o mouse nos ícones do **LinkedIn e GitHub**

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo

Exercitar conceitos fundamentais de **JavaScript**, como:

- Manipulação do DOM
- Eventos de clique (`onclick`)
- Eventos de mouse (`onmousemove`, `onmouseout`)
- Alteração dinâmica de imagens na página
